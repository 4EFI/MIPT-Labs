\begin{abstract}
    \textbf{ Цель работы: } 1) измерение объёмов форвакуумной и высоковакуумной частей установки; 2) определение скорости откачки системы в стационарном режиме, а также по ухудшению и по улучшению вакуума. \\ 
    
    \textbf{ В работе используются: } вакуумная установка с манометрами: масляным, термопарным и ионизационным. 
\end{abstract}

\section {Теоретическая справка} 
    Вакуумные установки делят на три типа: низковакуумные (\(10^{-2}-10^{-3} \ торр\)), высоковакуумные (\(10^{-4}-10^{-7} \ торр\)), установки сверхвысокого вакуума (\(10^{-8}-10^{-11} \ торр\)). Так же можно определить высокий вакуум как газ, в котором длина свободного пробега частиц сравнима с размерами установками.
    \\ 
    
    \textbf{ Форвакуумный насос}. В цилиндрической части насоса расположен ротор, который всегда соприкасается со своей верхней частью.
    \begin{figure}[h]
        \centering
        \includegraphics{6.png}
    \end{figure}
    \\
    На рисунке можно видеть, как воздух порциями гоняется из правой трубки в левую из-за вращения ротора.
    \\
    \\
    \textbf{ Диффузионный насос}. Нагревается масло, пары которого выходят через сопло. Значить частицы воздуха направляются в ту же сторону. Воздух выходит из системы, а масло конденсируется на стенках колбы (в виде елочки) и стекает обратно в начальное положение установки. При работе с таким насосом важно, чтоб давление в системе было достаточно низким, чтобы образовался поток масляных паров. Иначе масло просто будет выгорать. \\
    \\
    \begin{figure}[h]
        \centering
        \includegraphics[width=0.6\linewidth]{7.png}
    \end{figure}
    \\
    
    Скорость откачки диффузионного насоса намного выше скорости форвакуумного насоса, в чем нам предстоит убедиться.
    \\
    
    \textbf{ W} - это объем газа, который удаляется из системы за секунду при данном давлении. Таким образом, скорость откачки форвакуумного насоса -- это объем его воздуховой камеры, умноженный на частоту ротора. 
    
    Обозначим количество газа, которое приходит в трубку в процессе откачки за \(Q\). Тогда мы можем записать уравнение:
    \[-VdP = (PW - Q)dt\]
    Но при предельном давлении \(\frac{dP}{dt} = 0 \Rightarrow Q = P_{пр}W\). Подставим это в начальное уравнение и про интегрируем и, учитывая что \(P_0 \gg P_{пр}\):
    \[
        P - P_{пр} = P_0 e^{-Wt/V}
    \]
    
    Для течения газа через трубу при высоком вакууме справедлива формула:
    \[\frac{d(PV)}{dt} = \frac{4}{3}r^3\sqrt{\frac{2\pi RT}{\mu}} \frac{P_2 - P_1}{L}\]
    Если пренебречь давлением $P_1$ у конца, обращенного к насосу, получаем формулу для пропускной способности трубы:
    \[C_{тр} = \frac{dV}{dt} = \frac{4}{3}\frac{r^3}{L}\sqrt{\frac{2\pi RT}{\mu}}\]
    Для пропускной способности отверстия (например в кранах) имеем формулу:
    \[C_{отв}=S\frac{\bar v}{4}\]

\section {Результаты измерений и обработка данных} 

\subsection{ Определение объема форвакуумной и высоковакуумной частей установки }

    Возьмем данные с установки:  
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=0.6\linewidth]{1.jpg}
    \end{figure}

    Зная объем запертого воздуха $V_з = 50 см^3$ и плотность масла $\rho_м = 0,885 г/см^3$ найдем сначала объем форвакуумной части, пользуясь законом Бойля-Мариотта:
    \[
        V_{фв} = \dfrac{P_{атм} V_з }{ \rho_{м} g (h_{вв} - h_{вн}) }
    \]

    Где $h_{вв}, h_{вн}$ -- это соответственно высота масляного столба вверху и внизу. Чтобы получить давление в мм.масл.ст нам нужна разность данных высот. 

    Подставим числа и рассчитаем погрешность, зная что dh = 0,1 см.масл.ст:
    \[
        V_{фв} = 2259,2 \pm 8,9 см^3 
    \]

    Теперь рассчитаем объем общей части по такому же принципу:
    \[
        V_{общ} = 3536,2 \pm 21,9 см^3 
    \]

    Вычтя, найдем объем высоковакуумной части:
    \[
        V_{вв} = V_{общ} - V_{фв} = 1277,0 \pm 30,8 см^3  
    \]

\subsection{ Получение высокого вакуума и измерение скорости откачки }

    С помощью диффузионного насоса, принцип работы которого написан выше, получаем высокий вакуум. Теперь найдем скорость откачки по улучшению вакуума. Для этого отключаем откачку, перекрыв кран К3 и ждем ухудшения вакуума до примерно $8*10^{-4}$ торр, далее ежесекундно (после открытия крана К3) будем вносить данные в таблицу. Изобразим это на графике, предварительно прологарифмировав:

    \begin{figure}[H]
        \centering
        \includegraphics[width=0.65\linewidth]{4.png}
    \end{figure}

    Построив линию тренда, найдем коэффициент прямой. Зная что:
    \[
        \ln(P - P_{пр}) = -\dfrac{W}{V}t + \ln P_0
    \]

    найдем собственно скорость откачки:
    \[
        W = -k \times V = 0,302 \times 1277 = 385,7 \pm 12,4 см^3/с 
    \]

    Теперь проведем аналогичные действия для ухудшения вакуума:
    
    \begin{figure}[H]
        \centering
        \includegraphics[width=0.65\linewidth]{8.png}
    \end{figure}

    \[
        Q' = kV_{вв} = ( 36,7 \pm 0.8 ) \cdot 10^{-9} \  Дж \cdot с^{-1}
    \]
    \[
        Q = P_{пр}W = ( 46,2 \pm 1.3 ) \cdot 10^{-9} \ Дж \cdot с^{-1}
    \]

    Значит, \(Q_н = Q - Q' = ( 9,5 \pm 2.1 ) \cdot 10^{-9} \ Дж \cdot с^{-1}\)
